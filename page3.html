Modern Tetris

Modern Tetris is the newer version of Tetris that most platforms use.

Game mechanic Info
Tetromino Generator The tetrominoes come in a 7-bag randomizer. One of each tetromino comes every time in a different order.
Piece Preview Can have up to six piece previews, depending on the game
Lock Delay There is more delay until a piece locks in place when it reaches the stack
Rotation System SRS/Super Rotation System
Hold Option Option to hold a piece for future use
Hard Drop Allows the player to drop a piece immediately straight down
